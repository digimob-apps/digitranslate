# DigiTranslate

**Universal browser translation with learning mode, hover tooltips, glossary, and more.**

Available for Chrome and Firefox — free, no account required.

---

## Features

### Core Translation
- **Text selection** — select any text on a page and translate it instantly via popup or right-click context menu
- **Full page translation** — translate entire web pages in one click, with a live progress bar
- **Hover mode** — hover over any word to see an instant tooltip translation
- **Language auto-detection** — detects the page language from HTML metadata and adapts automatically

### Learning & Reference
- **Learning mode** — see both the original and translated text side-by-side to build vocabulary
- **Dictionary** — English definitions, phonetics, examples and synonyms via built-in lookup
- **Translation history** — last 100 translations stored locally, searchable, with page URL and timestamp
- **Bookmarks** — save translation pairs with tags and notes for later review

### Customization
- **Glossary** — define custom term pairs that override automatic translation for specific vocabulary
- **Domain blocklist** — disable the extension on specific sites
- **Keyboard shortcuts** — fully customizable hotkeys for all main actions
- **Compact mode** — reduced UI footprint for minimal visual distraction
- **Confidence scores** — optional display of translation confidence

### Text-to-Speech (TTS)
- **Browser TTS** — built-in speech synthesis, works offline, no setup required
- **ElevenLabs TTS** — high-quality neural voices via your own ElevenLabs API key (optional)

### Supported Languages
100+ languages via Google Translate, covering all major world languages.
